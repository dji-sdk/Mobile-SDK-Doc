      Package:dji.sdk.flightcontroller
      SDK Key:LANDING_GEAR_AUTOMATIC_MOVEMENT_ENABLED
      Doc Key:DJILandingGear_getAutomaticMovementEnabled
      Description:

      Indicates whether the self-adaptive landing gear is turned on or off. If the self-adaptive landing gear is turned on, when the drone is 0.5m above the ground, the landing gear will automatically lower. After it takes off, once the drone is 0.5m above the ground, the landing gear will automatically be raised up. If self-adaptive landing gear is turned off, the drone will not automatically lower and raise the landing gear.

      Input Parameters:
      @NonNull CompletionCallbackWith<Boolean> callbackCallback to receive the result.

      Current state of the Landing Gear.

      DEPLOYEDLanding Gear is fully deployed (ready for landing)
      RETRACTEDLanding Gear is fully retracted (ready for flying).

      LandingGearMode
      enum LandingGearMode
      Package:dji.common.flightcontroller
      Doc Key:DJILandingGear_DJILandingGearMode
      Description:

      Current Mode of the Landing Gear.

      Enum Members:
      MANUALLanding Gear can be deployed and retracted through function calls.
      TRANSPORTLanding Gear is in transport mode (either it is moving into, moving out of, or stopped in transport position).
      AUTOLanding Gear automatically transitions between deployed and retracted depending on altitude. During take-off, the transition point is 1.2m above ground. After take-off (during flight or when landing), the transition point is 0.5m above ground.
      UNKNOWNLanding Gear is in an unknown mode.
